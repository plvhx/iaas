### IaaS (Insult as a Service)

Clone:

```
$ git clone https://github.com/plvhx/iaas.git
```

Install deps:

```
$ cd iaas
$ npm install
```

Get your insult of the day:

```
$ curl -X GET http://localhost:3000/insult
$ {"id":19,"content":"Boy, sit your $5 dollar ass down before I make change!"}
```
